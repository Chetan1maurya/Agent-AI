# Agent-AI
This is just react based frontend project using framer motion animation and tailwindcss.
Tech Stack:
1). React based 
2). Tailwindcss for designing
3). Framer motion for animations
This project is for understanding how to use framer motion, tailwindcss and orgainize each react components.

## Run Locally

git clone https://github.com/chetan1maurya/Agent-AI.git
cd Agent-AI
npm install
npm run dev
